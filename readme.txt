Nama Aplikasi : Amalaundry

yang perlu disiapkan: 
 1. xampp
 2. npm
 
cara menjalankan aplikasi
 1. nyalakan webserver dan database dalam xampp
 2. buka cmd lalu di direct ke folder aplikasi
 3. ketikkan perintah php artisan migration:refresh --seed
 3. lalu ketikkan perintah "npm run dev"
 4. setelahnya ketikkan perintah "php artisan serve"
 6. buka aplikasi di browser dengan url 127.0.0.1:8000
 7. untuk login, email:superadmin@gmail.com pass:admin
